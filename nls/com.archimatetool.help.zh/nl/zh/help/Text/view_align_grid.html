对齐工具、参考线和网格

为了方便绘制美观的图表，提供了各种对齐工具。这些可从主“视图->位置”菜单项、主工具栏或右键单击图表的画布区域以选择“位置”菜单项。它们如下：

放大/缩小/实际尺寸： 放大和缩小图表。这也可以在工具栏上作为组合框使用。您还可以通过按住 Ctrl 键并滚动鼠标滚轮来放大和缩小。
对齐网格： 将元素捕捉到网格，无论它是否可见。 （网格间距可以在首选项中设置。）
网格可见： 切换网格的可见性。 （网格间距可以在首选项中设置。）
对齐对齐参考线： 这些是在图表中拖动元素时出现的蓝色对齐参考线（线）。它们有助于排列元素的边缘和中心。
左对齐： 当两个或更多元素被选中时，在主选区的左边缘对齐。
中心对齐： 当两个或更多元素被选中时，与主选区水平居中对齐。
右对齐： 当两个或更多元素被选中时，在主选区的右边缘对齐。
上对齐： 当两个或多个元素被选中时，在主选区的上边缘对齐。
居中对齐： 当两个或更多元素被选中时，垂直居中对齐到主选区。
底部对齐： 当两个或更多元素被选中时，在主选区的底部边缘对齐。
匹配宽度： 当两个或更多元素被选中时，元素的宽度与主选择相匹配。
匹配高度： 选择两个或多个元素时与元素的高度匹配到主要选择。
匹配大小： When two or more elements are selected match the height and width of the elements to the primary selection.
默认尺寸： 将所选元素设置为其默认大小。如果禁用，则元素已设置为其默认大小。

有关主要选择的信息，请参阅调色板选择工具。
